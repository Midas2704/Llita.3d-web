# Llita.3D Web

Landing page estatica para Llita.3D, emprendimiento de impresion 3D personalizada.

## Vista local

```bash
python -m http.server 5173
```

Luego abrir:

```text
http://127.0.0.1:5173/
```

## Pendientes de personalizacion

- Reemplazar el numero de WhatsApp en `index.html`.
- Reemplazar redes sociales.
- Cambiar placeholders de galeria por fotos reales.
